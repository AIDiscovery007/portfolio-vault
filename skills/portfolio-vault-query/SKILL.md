# Portfolio Vault Query

Use read-only Portfolio Vault MCP tools first:

- `get_portfolio_summary`
- `list_import_drafts`
- `get_instrument_registry`

Treat Portfolio Vault as a recordkeeping and analysis tool. Do not give buy/sell recommendations or execute trades. If the user asks for investment advice, keep the answer limited to factual portfolio structure, data gaps, concentration, costs, cash, and reconciliation observations.
